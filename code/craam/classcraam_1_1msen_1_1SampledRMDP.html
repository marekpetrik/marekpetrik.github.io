Detailed Description

template<typename Model>
class craam::msen::SampledRMDP< Model >

Constructs a robust MDP from integer samples.

In integer samples each decision state, expectation state, and action are identified by an integer.

There is some extra memory penalty in this class over a plain MDP since it stores the number of samples observed for each state and action.

Important: Actions that are not sampled (no samples per that state and action pair) are labeled as invalid and are not included in the computation of value function or the solution.
